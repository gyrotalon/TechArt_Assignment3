# Batch File Renamer

## Description

This program launches a GUI for a batch file renamer that can rename files or create a copy of the file with a new file name.
This was an assignment for my technical art class.

## Getting Started

### Dependencies

PyQt6
This has only been tested on Windows 10 with Python 3.13
